## 1. 项目概述

### 1.1 项目简介

AI宠物健康助手是一个基于LangChain框架开发的智能系统，旨在为宠物主人提供全面的宠物健康管理服务。系统结合了先进的AI技术和专业的宠物医疗知识，能够回答宠物健康问题、提供健康建议、跟踪健康状况并连接宠物主人与专业兽医。

### 1.2 项目目标

- 为宠物主人提供便捷的宠物健康咨询服务
- 帮助宠物主人识别宠物健康问题的早期信号
- 提供个性化的宠物健康管理方案
- 促进宠物主人与兽医之间的有效沟通
- 建立宠物健康数据档案，实现长期健康跟踪

### 1.3 技术基础

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 后端框架 | Python 3.12.10 + FastAPI 0.135.3 | 高性能异步API框架 |
| 数据库 | MySQL 8.0+ / SQLite(测试) | 关系型数据存储 |
| ORM | SQLAlchemy 2.0 | 数据库抽象层 |
| AI框架 | LangChain + ChromaDB | 向量数据库+Agent编排 |
| 认证 | JWT(HS256) + bcrypt | 无状态认证+密码哈希 |
| 前端 | Next.js 14 + TypeScript + Zustand + Tailwind CSS | 已完成 |
| 测试 | pytest + Jest + Playwright | 多层测试体系 |

## 3. 系统架构

### 3.1 架构设计

系统采用模块化、分层架构设计，主要包括以下层次：

1. **用户界面层**：Next.js 14 Web应用，支持文本交互
2. **应用服务层**：FastAPI处理用户请求，协调各功能模块
3. **核心功能层**：实现健康咨询、症状分析、行为分析等核心功能
4. **知识处理层**：管理宠物健康知识图谱和ChromaDB向量数据库
5. **外部集成层**：与LLM API（DeepSeek/GPT）等外部系统集成
6. **数据存储层**：MySQL存储结构化数据，ChromaDB存储语义向量
