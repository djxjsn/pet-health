# Trae MCP 快速开始指南

**5 分钟快速配置 MCP 工具**

---

## 🚀 快速配置步骤

### 步骤 1: 打开 Trae IDE (30 秒)

```
1. 启动 Trae IDE
2. 打开项目：e:\学习\trae project\pei health agent
```

### 步骤 2: 进入 MCP 设置 (30 秒)

```
1. 点击左侧导航栏的 "MCP" 图标
2. 点击右上角的 "添加" 按钮
3. 选择 "手动添加"
```

### 步骤 3: 分别导入三个配置文件 (3 分钟)

⚠️ **重要**: Trae 一次只能添加一个 MCP Server，请依次导入以下三个配置文件

#### 3.1 添加 Plan / Workflow MCP

1. 打开文件：`.trae/mcp-settings-01-plan.json`
2. 全选并复制所有内容 (Ctrl+A, Ctrl+C)
3. 粘贴到 Trae MCP 配置窗口
4. 点击 "保存"

⚠️ **重要**: 配置后需要修改 Token：
- 编辑 `.trae/mcp-settings-01-plan.json`
- 将 `DEVGENIUS_MCP_TOKEN` 替换为你的实际 Token

#### 3.2 添加 Memory / Vector DB MCP

1. 再次点击右上角 "添加" 按钮
2. 打开文件：`.trae/mcp-settings-02-memory.json`
3. 全选并复制所有内容
4. 粘贴到 Trae MCP 配置窗口
5. 点击 "保存"

#### 3.3 添加 Skill / Function Hub MCP

1. 再次点击右上角 "添加" 按钮
2. 打开文件：`.trae/mcp-settings-03-skill.json`
3. 全选并复制所有内容
4. 粘贴到 Trae MCP 配置窗口
5. 点击 "保存"

### 步骤 4: 启用 MCP Server (30 秒)

```
在 MCP 设置面板中：
✓ 启用 plan-workflow
✓ 启用 memory-vector
✓ 启用 skill-hub
```

### 步骤 5: 测试 MCP 工具 (2 分钟)

在 Trae AI 对话框中测试：

**测试 Plan MCP**:
```
查看我的任务列表
```

**测试 Memory MCP**:
```
记住这个项目使用 FastAPI 框架
```

**测试 Skill Hub MCP**:
```
列出 skills 目录中的文件
```

## 🔧 获取 Plan MCP Token

**如已有 Token，跳过此步骤**

1. 访问 DevGenius 平台（需搜索官网）
2. 注册账号
3. 在个人设置中生成 MCP Token
4. 复制 Token（格式：`mcp_xxxxxxxxxx`）
5. 编辑 `.trae/mcp-settings.json`
6. 替换 `DEVGENIUS_MCP_TOKEN` 为你的实际 Token
7. 保存并重启 Trae IDE

## ❓ 常见问题

### Q: Plan MCP 无法启动？
**A**: 检查是否配置了有效的 Token

### Q: MCP Server 下载很慢？
**A**: 配置 npm 镜像：
```powershell
npm config set registry https://registry.npmmirror.com
```

### Q: 配置导入失败？
**A**: 尝试手动逐个添加 MCP Server

### Q: MCP 工具无响应？
**A**: 
1. 检查 Trae 日志
2. 重启 Trae IDE
3. 禁用后重新启用 MCP Server
