## 四、遇到的问题与解决方案

### 4.1 问题一：数据库迁移冲突

- **问题描述**：迁移文件编号冲突（已存在 004 文件）
- **解决方案**：检查现有迁移文件，确认功能完整性

### 4.2 问题二：宠物归属校验

- **问题描述**：需要确保用户只能操作自己的宠物
- **解决方案**：在 API 层校验 `pet.user_id == current_user.user_id`

### 4.3 问题三：外部 LLM 依赖

- **问题描述**：健康分析需要调用外部 LLM API
- **解决方案**：使用 Mock 对象进行测试，确保测试稳定性
