# DEV-007 智能购物助手模块 - 技术设计文档

## 1. 模块概述

### 1.1 功能定位

智能购物助手是 AI 宠物健康助手系统的核心功能模块之一，为用户提供：

- **智能商品搜索**：基于关键词、分类、价格等多维度搜索
- **成分安全分析**：分析宠物食品成分的安全性和营养价值
- **个性化推荐**：结合用户历史和宠物健康状态的智能推荐
- **购物行为追踪**：记录用户浏览、加购、购买等行为
- **商品对比功能**：多维度对比不同商品的特性

### 1.2 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 框架 | FastAPI | 高性能异步Web框架 |
| 数据库 | MongoDB | 文档型数据库，灵活的Schema |
| 缓存 | Redis (预留) | 热门商品缓存 |
| AI集成 | RAG知识库 | 成分分析和推荐算法 |
| 测试框架 | pytest + pytest-asyncio | 单元测试/集成测试 |

### 2.2 核心组件职责

#### ProductSearcher（商品搜索引擎）

**文件**: `src/core/product_searcher.py`

**职责**:
- 关键词模糊搜索
- 分类筛选
- 价格范围过滤
- 排序（评分/价格/时间）
- 相似商品推荐（±30%价格区间）

**核心方法**:

| 方法 | 说明 |
|------|------|
| `search()` | 多条件综合搜索 |
| `search_by_category()` | 按分类列表查询 |
| `get_product_details()` | 获取商品详情 |
| `get_similar_products()` | 获取相似商品 |

#### RecommendationEngine（推荐引擎）

**文件**: `src/core/recommendation_engine.py`

**职责**:
- 基于用户历史的个性化推荐
- 基于RAG的知识导向推荐
- 针对健康问题的专项推荐
- 热门商品兜底推荐
- 多源结果合并去重

**推荐策略优先级**:

```
个性化推荐 > 知识导向推荐 > 热门商品 > 健康导向推荐
```

#### IngredientAnalyzer（成分分析器）

**文件**: `src/core/ingredient_analyzer.py`

**职责**:
- 成分文本解析
- 安全性评估（safe/cautionary/unsafe）
- 过敏原检测
- 营养价值分析
- 针对不同宠物类型的适配评估

**安全等级定义**:

| 等级 | 分数范围 | 行为建议 |
|------|---------|---------|
| safe | 80-100 | 可放心使用 |
| cautionary | 50-79 | 谨慎使用，关注特定成分 |
| unsafe | 0-49 | 不建议使用，存在风险 |

## 3. 数据模型

### 3.1 商品模型 (Product)

```python
class Product(BaseModel):
    _id: Optional[str] = None
    product_id: str                    # 业务ID，格式: product_{category}_{timestamp}
    name: str                          # 商品名称
    category: str                      # 分类: food/toy/accessory/medicine/hygiene/clothing
    price: float                       # 价格
    brand: Optional[str] = None        # 品牌
    subcategory: Optional[str] = None  # 子分类
    image_url: Optional[str] = None    # 图片URL
    description: Optional[str] = None  # 描述
    rating: float = 0.0                # 评分(0-5)
    stock_status: str = "in_stock"     # 库存状态
    currency: str = "CNY"              # 货币单位
    
    # 扩展字段
    ingredients: List[Dict] = []       # 成分列表
    nutrition_info: Dict = {}          # 营养信息
    suitable_for: List[str] = []       # 适用对象
    tags: List[str] = []               # 标签
    review_count: int = 0              # 评论数量
    
    created_at: datetime = Field(default_factory=datetime.utcnow)
    updated_at: datetime = Field(default_factory=datetime.utcnow)
```

### 3.2 购物历史模型 (ShoppingHistory)

```python
class ShoppingHistory(BaseModel):
    history_id: str                    # 格式: history_{timestamp}
    user_id: str                       # 用户ID
    product_id: str                    # 商品ID
    action_type: str                   # search/view/cart/purchase/wishlist
    pet_id: Optional[str] = None       # 关联宠物
    price_at_time: float               # 快照价格
    search_query: Optional[str] = None # 搜索关键词
    created_at: datetime = Field(default_factory=datetime.utcnow)
```

### 3.3 成分分析结果模型 (IngredientAnalysis)

```python
class IngredientAnalysis(BaseModel):
    analysis_id: str                   # 分析记录ID
    ingredients_text: str              # 原始成分文本
    pet_type: str                      # 宠物类型
    overall_safety: str                # 整体安全性
    safety_score: int                  # 安全分数(0-100)
    safe_ingredients: List[Dict]       # 安全成分
    caution_ingredients: List[Dict]    # 需注意成分
    unsafe_ingredients: List[Dict]     # 不安全成分
    allergen_warnings: List[Dict]      # 过敏原警告
    recommendations: List[str]         # 建议
    total_analyzed: int                # 总分析数
    parsed_ingredients: List[str]      # 解析后的成分列表
    created_at: datetime = Field(default_factory=datetime.utcnow)
```

## 5. 关键业务流程

### 5.1 商品搜索流程

```
用户输入 → API接收参数 → ProductSearcher.search()
                              ↓
                     参数校验与构建
                              ↓
                     ProductRepository.search()
                              ↓
                     结果格式化返回
```

### 5.2 推荐生成流程

```
用户请求 → RecommendationEngine.get_recommendations()
                    ↓
    ┌───────────────┼───────────────┐
    ↓               ↓               ↓
个性化推荐    知识导向推荐    热门商品推荐
    ↓               ↓               ↓
    └───────────────┼───────────────┘
                    ↓
          _merge_and_deduplicate()
                    ↓
          按优先级合并去重
                    ↓
              返回推荐结果
```

### 5.3 成分分析流程

```
成分文本输入 → IngredientAnalyzer.analyze()
                    ↓
          _parse_ingredients()
                    ↓
          逐个成分安全检查
                    ↓
    ┌───────────┼───────────┐
    ↓           ↓           ↓
 安全成分   注意成分   危险成分
    ↓           ↓           ↓
过敏原检测  综合评分计算
    ↓           ↓           ↓
    └───────────┼───────────┘
                ↓
          生成建议和报告
```
