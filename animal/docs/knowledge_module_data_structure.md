智能宠物健康知识科普模块 - 数据结构定义

本文件为宠物品种百科和健康知识科普模块定义了完整的数据结构，涵盖猫、狗两大类别的常见品种信息及常见病症数据库。数据作为静态知识库，由服务层加载提供，以确保知识内容的准确性和可维护性。

## 数据内容总览

### 品种百科（共20个品种）
- 猫（10个品种）：中华田园猫、布偶猫、英国短毛猫、美国短毛猫、暹罗猫、波斯猫、苏格兰折耳猫、缅因猫、异国短毛猫(加菲猫)、斯芬克斯猫(无毛猫)
- 狗（10个品种）：中华田园犬、金毛寻回犬、拉布拉多寻回犬、柯基犬、贵宾犬(泰迪)、哈士奇、萨摩耶犬、边境牧羊犬、柴犬、德国牧羊犬

### 健康知识（按物种和病症类别分类）
- 猫常见病症：消化系统、呼吸系统、皮肤系统、泌尿系统、眼部疾病、口腔疾病、寄生虫、传染病
- 狗常见病症：消化系统、呼吸系统、皮肤系统、骨骼关节、眼部疾病、口腔疾病、寄生虫、传染病

## 数据文件说明

| 文件路径 | 内容 | 格式 |
|---------|------|------|
| animal/src/services/encyclopedia_service.py | 完整的品种和健康知识数据 + 检索服务 | Python dict |

## 数据更新机制

- 知识数据采用硬编码方式内置于服务层
- 如需更新，修改 encyclopedia_service.py 中的 BREEDS_DATA 和 HEALTH_DATA 字典
- 更新后重新部署服务即可生效
- 频率：预计每季度更新一次

## 数据准确性声明

- 品种信息参考自：CKU（中国犬业俱乐部）、CFA（国际爱猫联合会）标准
- 健康知识参考自：兽医临床手册、兽医药典
- 所有健康建议仅供参考，不能替代专业兽医诊断
