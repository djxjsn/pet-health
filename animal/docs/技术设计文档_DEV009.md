# DEV-009 数据安全与隐私模块 - 技术设计文档

## 1. 模块概述

### 1.1 功能定位

DEV-009 是 AI 宠物健康助手系统的数据安全基础设施模块，提供：

- **数据加密存储**: AES-256-GCM 对称加密
- **敏感数据脱敏**: 9种内置脱敏策略
- **RBAC权限控制**: 5级角色 + 14项权限
- **审计日志追踪**: 不可变操作记录
- **安全API管理**: 10个管理端点

### 1.2 技术栈

| 组件 | 技术 | 说明 |
|------|------|------|
| 加密算法 | AES-256-GCM | 认证加密，防篡改 |
| 密钥派生 | PBKDF2-SHA256 | 10万次迭代 |
| 脱敏引擎 | 自研 | 9种策略 |
| 权限模型 | RBAC | 基于角色的访问控制 |
| 审计存储 | MongoDB | 仅追加，不可变 |
| API框架 | FastAPI | 依赖注入权限检查 |

## 3. 核心组件详解

### 3.1 EncryptionService（加密服务）

**算法**: AES-256-GCM  
**密钥派生**: PBKDF2-SHA256 (100,000次迭代)

| 方法 | 说明 |
|------|------|
| `encrypt(plaintext)` | 加密字符串 |
| `decrypt(encrypted_text)` | 解密字符串 |
| `encrypt_field(value, field_name)` | 加密数据库字段 |
| `decrypt_field(encrypted, field_name)` | 解密数据库字段 |
| `generate_key()` | 生成随机密钥 |
| `hash_data(data)` | 不可逆哈希 |

**安全特性**:
- 每次加密生成随机12字节nonce
- 使用关联数据(AAD)绑定字段名，防止字段替换攻击
- Base64编码存储
- 解密时自动验证认证标签，检测篡改

### 3.2 DataMasker（脱敏服务）

| 策略 | 示例输入 | 示例输出 |
|------|---------|---------|
| phone | 13800138000 | 138****8000 |
| email | user@example.com | u***@example.com |
| id_card | 110101199001011234 | 110***********1234 |
| name | 张三丰 | 张*丰 |
| address | 北京市朝阳区某某路123号 | 北京市朝阳区**** |
| bank_card | 6222021234567890123 | **** **** **** 0123 |
| password | MySecret123! | ****** |
| ip_address | 192.168.1.100 | 192.168.*.* |
| custom | ABCDEFGHIJ(前3后2) | ABC*****IJ |

**批量脱敏**: `mask_dict()` / `mask_nested_dict()` 支持字典和嵌套结构

### 3.3 RBAC权限控制

**角色层级**:

```
SUPER_ADMIN (14权限) → ADMIN (16权限) → VETERINARIAN (8权限) → USER (6权限) → GUEST (3权限)
```

**权限清单**:

| 类别 | 权限代码 | 说明 |
|------|---------|------|
| 用户 | user:read/write/delete | 用户管理 |
| 宠物 | pet:read/write/delete | 宠物管理 |
| 健康 | health:read/write | 健康咨询 |
| 购物 | shop:read/write | 购物功能 |
| 知识 | knowledge:read/write | 知识库 |
| 管理 | admin:panel | 管理面板 |
| 审计 | audit:read | 审计日志 |
| 安全 | security:config | 安全配置 |
| 数据 | data:export/encryption | 数据操作 |

**FastAPI集成**:
```python
# 权限检查
@router.get("/admin", dependencies=[Depends(require_permission(Permission.ADMIN_PANEL))])

# 角色检查
@router.get("/admin", dependencies=[Depends(require_role(Role.ADMIN))])
```

### 3.4 AuditLogService（审计日志）

**存储**: MongoDB `audit_logs` 集合  
**特性**: 仅追加，不可修改/删除

**日志类型**:

| 类别 | 操作 |
|------|------|
| 认证 | login/logout/login_failed/token_refresh/password_change |
| 用户 | create/update/delete/read |
| 宠物 | create/update/delete |
| 健康 | consult/record_create/record_update |
| 购物 | search/purchase |
| 安全 | data_export/encrypt/decrypt/permission_change/role_change |
| 系统 | config/error |

**日志级别**: INFO → WARNING → CRITICAL
