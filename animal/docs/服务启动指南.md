## 🔧 常用命令

### 启动服务
```bash
cd animal
python -m uvicorn src.main:app --reload
```

### 启动服务（带参数）
```bash
# 指定端口
python -m uvicorn src.main:app --reload --port 8001

# 指定主机
python -m uvicorn src.main:app --reload --host 0.0.0.0

# 生产模式（不自动重载）
python -m uvicorn src.main:app --workers 4
```

### 停止服务
```
按 Ctrl+C
```

### 检查服务状态

## 🔍 故障排查

### 问题 1: 服务无法启动
```
症状：端口被占用
解决：
  # 查看占用端口的进程
  netstat -ano | findstr :8000
  
  # 或使用不同端口
  python -m uvicorn src.main:app --reload --port 8001
```

### 问题 2: 数据库连接失败
```
症状：Database connection error
解决：
  1. 检查 MySQL 服务是否启动
  2. 检查 .env 配置
  3. 确认数据库已创建
```

### 问题 3: APIFox 无法访问本地服务
```
症状：Connection refused
解决：
  1. 确保服务监听 0.0.0.0
  2. 检查防火墙设置
  3. 使用客户端版 APIFox
```
