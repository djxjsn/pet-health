# 宠物营养建议

## 基本营养需求

### 六大营养素

1. **蛋白质**
   - 作用：构建和修复组织、产生抗体
   - 来源：肉类、鱼类、蛋类
   - 需求量：犬18-25%，猫26-30%

2. **脂肪**
   - 作用：提供能量、维持皮肤和毛发健康
   - 来源：动物脂肪、鱼油
   - 需求量：犬10-15%，猫20-24%

3. **碳水化合物**
   - 作用：提供能量、促进消化
   - 来源：米饭、薯类、蔬菜
   - 需求量：适量即可，犬猫非必需

4. **维生素**
   - 作用：维持正常生理功能
   - 种类：A、D、E、K、B族维生素
   - 来源：均衡饮食通常可满足

5. **矿物质**
   - 作用：骨骼发育、神经传导
   - 重要矿物质：钙、磷、铁、锌
   - 比例：钙磷比1.2:1

6. **水**
   - 作用：维持生命活动
   - 需求量：每千克体重每天约50-100ml
   - 建议：提供充足新鲜饮水

## 不同阶段营养需求

### 幼年期（0-12月龄）

**特点**：快速生长发育

**营养重点**：
- 高蛋白（支持生长）
- 高能量
- 充足的钙和磷（骨骼发育）
- DHA（大脑和视力发育）

**喂食建议**：
- 每日喂食3-4次
- 使用幼犬/幼猫专用粮
- 不要过度喂食（避免肥胖）
- 逐渐过渡到成犬粮（10-12月龄）

### 成年期（1-7岁）

**特点**：维持健康体重和活力

**营养重点**：
- 均衡营养
- 控制热量（防止肥胖）
- 优质蛋白质
- 充足纤维（促进消化）

**喂食建议**：
- 每日喂食2次
- 定时定量
- 根据活动量调整
- 定期评估体重和身体状况

### 老年期（7岁以上）

**特点**：代谢下降，器官功能衰退

**营养重点**：
- 降低热量（防止肥胖）
- 优质易消化蛋白质
- 关节保护（葡萄糖胺、软骨素）
- 抗氧化剂（延缓衰老）
- Omega-3脂肪酸（抗炎）

**喂食建议**：
- 使用老年犬/猫专用粮
- 每日喂食2-3次，少量多餐
- 定期体检
- 关注肾脏和牙齿健康

## 特殊需求营养

### 减肥饮食

**适用情况**：体重超标10%以上

**饮食调整**：
- 降低热量摄入（减少20-30%）
- 高纤维（增加饱腹感）
- 低脂肪
- 优质蛋白质（保持肌肉）

**喂食策略**：
- 定时定量
- 减少零食
- 增加运动
- 每周监测体重

### 过敏饮食

**常见过敏原**：
- 牛肉
- 乳制品
- 小麦
- 鸡肉
- 鸡蛋
- 大豆

**饮食管理**：
- 确定过敏原
- 使用低敏配方粮
- 尝试新型蛋白质（鹿肉、鸭肉）
- 避免人类食物

### 肾脏疾病饮食

**营养调整**：
- 降低蛋白质（减轻肾脏负担）
- 降低磷（延缓病情）
- 增加Omega-3（抗炎）
- 限制钠（控制血压）

**注意事项**：
- 必须遵医嘱
- 使用处方粮
- 定期监测肾功能

## 食物安全

### 有害食物清单

#### 犬类禁忌

| 食物 | 危害 |
|------|------|
| 巧克力 | 可可碱中毒，影响心脏和神经系统 |
| 葡萄/葡萄干 | 肾衰竭 |
| 洋葱/大蒜 | 破坏红血球，导致贫血 |
| 木糖醇 | 低血糖、肝衰竭 |
| 酒精 | 中枢神经系统抑制 |
| 咖啡/茶 | 咖啡因中毒 |
| 澳洲坚果 | 肌肉无力、呕吐 |
| 生面团 | 肠胃膨胀、酒精中毒 |

#### 猫类禁忌

| 食物 | 危害 |
|------|------|
| 洋葱/大蒜 | 破坏红血球，导致贫血 |
| 巧克力 | 可可碱中毒 |
| 咖啡/茶 | 咖啡因中毒 |
| 酒精 | 中枢神经系统抑制 |
| 生鱼 | 破坏维生素B1 |
| 牛奶 | 乳糖不耐受 |
| 百合花 | 急性肾衰竭 |
| 葡萄/葡萄干 | 可能有害 |

### 喂食注意事项

1. **逐渐换粮**：7-10天过渡期
   - 第1-2天：25%新粮
   - 第3-4天：50%新粮
   - 第5-6天：75%新粮
   - 第7天：100%新粮

2. **不要喂食**：
   - 变质食物
   - 过期食品
   - 人类剩菜剩饭
   - 骨头（特别是禽骨）

3. **零食原则**：
   - 不超过每日总热量的10%
   - 选择健康零食
   - 避免高盐、高糖

## 喂食量计算

### 每日能量需求（DER）

**公式**：
```
RER（静息能量需求）= 70 × 体重(kg)^0.75
DER = RER × 活动系数
```

**活动系数**：
- 绝育成犬：1.6
- 未绝育成犬：1.8
- 老年犬：1.4
- 减肥犬：1.0
- 工作犬：2.0-5.0
- 幼犬：2.0-3.0
- 怀孕犬：1.8-3.0

### 喂食量参考

**小型犬（5kg）**：
- 成犬：约100-150g/天
- 幼犬：约150-200g/天

**中型犬（15kg）**：
- 成犬：约250-350g/天
- 幼犬：约350-450g/天

**大型犬（30kg）**：
- 成犬：约400-550g/天
- 幼犬：约500-700g/天

*注：具体喂食量需根据狗粮包装说明和宠物实际情况调整*
