# 宠物急救手册

## 急救基本原则

### 1. 保持冷静

- 评估情况，判断紧急程度
- 安抚宠物，避免恐慌
- 安全接近，防止咬伤

### 2. 初步评估

**检查生命体征**：
- 呼吸频率和深度
- 心跳（正常犬：60-140次/分，猫：140-220次/分）
- 体温（正常犬：37.5-39.2°C，猫：38.0-39.2°C）
- 意识状态
- 出血情况

### 3. 紧急联系

- 立即联系最近的宠物医院
- 说明症状和紧急程度
- 按照兽医指导进行初步处理

## 常见急救情况

### 窒息急救

**症状**：
- 呼吸困难或停止
- 抓挠嘴部
- 牙龈发绀（蓝紫色）
- 恐慌不安

**急救步骤**：
1. 小心打开口腔
2. 检查是否有异物阻塞
3. 如能看到异物，用钳子或手指取出
4. 如看不到，尝试海姆立克急救法：
   - 小型犬/猫：倒提并轻拍背部
   - 大型犬：从背后抱住，在肋骨下方快速向上挤压
5. 立即就医

### 出血急救

**轻微出血**：
1. 用干净纱布按压伤口
2. 持续按压5-10分钟
3. 清洁伤口
4. 包扎固定

**严重出血**：
1. 立即按压出血点
2. 抬高出血部位（如可能）
3. 使用绷带加压包扎
4. 不要移除已经浸血的绷带，直接在上面加层
5. 立即就医

**止血带使用**（仅在危及生命时）：
- 绑在伤口上方（靠近心脏侧）
- 每15-20分钟放松1-2分钟
- 记录使用时间
- 尽快就医

### 中毒急救

**常见中毒源**：
- 巧克力（犬）
- 葡萄/葡萄干（犬）
- 洋葱/大蒜（犬猫）
- 木糖醇（犬）
- 杀虫剂
- 老鼠药
- 人类药物
- 某些植物（百合对猫剧毒）

**中毒症状**：
- 呕吐、腹泻
- 流口水
- 抽搐、震颤
- 意识不清
- 呼吸困难
- 心跳异常

**急救步骤**：
1. 确定中毒物质（带上包装）
2. 联系兽医毒物控制中心
3. 不要自行催吐（除非兽医指导）
4. 如皮肤接触，立即用大量清水冲洗
5. 如吸入毒气，移至空气新鲜处
6. 立即就医

### 烧烫伤急救

**症状**：
- 皮肤红肿
- 水泡
- 脱毛
- 疼痛

**急救步骤**：
1. 立即用冷水冲洗15-20分钟
2. 不要用冰块直接敷
3. 不要涂抹药膏、黄油等
4. 用干净湿布覆盖
5. 立即就医

### 骨折急救

**症状**：
- 肢体变形
- 无法承重
- 肿胀
- 疼痛

**急救步骤**：
1. 限制宠物活动
2. 不要试图复位
3. 如有开放性骨折，用干净纱布覆盖
4. 使用夹板固定（如可能）
5. 搬运时保持平稳
6. 立即就医

### 中暑急救

**症状**：
- 体温超过40°C
- 急促喘气
- 流口水
- 呕吐
- 意识不清
- 站立不稳

**急救步骤**：
1. 立即移至阴凉处
2. 用温水（非冰水）淋湿身体
3. 用风扇加速散热
4. 提供饮水（不要强迫）
5. 监测体温，降至39.5°C停止降温
6. 立即就医

### 癫痫发作

**症状**：
- 抽搐
- 失去意识
- 排尿/排便
- 口吐白沫

**急救步骤**：
1. 保持冷静，计时
2. 清除周围危险物品
3. 不要将手放入宠物口中
4. 不要试图按压或制止抽搐
5. 降低刺激（关灯、安静）
6. 抽搐超过5分钟立即就医
7. 记录发作时间和症状

### 车祸急救

**急救步骤**：
1. 确保安全后再接近
2. 小心移动宠物（可能咬人）
3. 评估意识和呼吸
4. 止血
5. 使用硬板搬运
6. 保持头部和脊柱稳定
7. 立即就医

## CPR心肺复苏

### 适应症
- 呼吸停止
- 心跳停止
- 意识完全丧失

### 操作步骤

1. **评估**：
   - 检查呼吸和心跳
   - 确认无意识

2. **人工呼吸**：
   - 闭合宠物嘴巴
   - 对着鼻孔吹气
   - 看到胸部鼓起即可
   - 频率：每5秒一次

3. **胸外按压**：
   - 小型犬/猫：用拇指和手指按压
   - 大型犬：用双手交叠按压
   - 位置：胸部最宽处
   - 深度：胸部的1/3-1/2
   - 频率：100-120次/分

4. **比例**：
   - 30次按压 : 2次人工呼吸
   - 每2分钟评估一次

5. **持续进行**：
   - 直到恢复自主呼吸
   - 或到达医院
