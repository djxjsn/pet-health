# 宠物行为训练指南

## 常见行为问题

### 拆家行为

#### 原因分析

**生理因素**：
- 精力过剩
- 换牙期不适（幼犬）
- 营养不足

**心理因素**：
- 分离焦虑
- 无聊
- 寻求关注
- 压力或恐惧

**品种特性**：
- 哈士奇、金毛等高能量品种
- 猎犬类（需要大量运动）

#### 解决方案

1. **充足运动**：
   - 每天至少1-2小时户外活动
   - 跑步、接球、游泳
   - 消耗多余精力

2. **环境丰富**：
   - 提供耐咬玩具
   - 使用益智玩具（藏食玩具）
   - 定期更换玩具

3. **训练方法**：
   - 正面强化（奖励正确行为）
   - 提供合适的咀嚼物
   - 发现拆家时立即制止
   - 不要事后惩罚（无效）

4. **逐步训练**：
   - 短时间离开，逐渐增加
   - 离开前充分运动
   - 留下有你气味的物品

### 过度吠叫

#### 原因分析

- 警戒行为
- 分离焦虑
- 寻求关注
- 无聊
- 恐惧
- 身体不适

#### 解决方案

1. **确定原因**：
   - 记录吠叫场景
   - 观察触发因素

2. **针对性训练**：
   - 警戒吠叫：降低刺激
   - 焦虑吠叫：脱敏训练
   - 寻求关注：忽略（不强化）

3. **训练"安静"指令**：
   - 等宠物停止吠叫瞬间
   - 立即说"安静"并奖励
   - 重复训练

4. **避免**：
   - 大声呵斥（可能被理解为加入）
   - 体罚（增加焦虑）

### 攻击行为

#### 类型

1. **恐惧攻击**：
   - 表现：被逼入角落时攻击
   - 原因：恐惧、缺乏安全感

2. **领地攻击**：
   - 表现：保护食物、玩具、空间
   - 原因：领地意识

3. **疼痛攻击**：
   - 表现：触碰某部位时攻击
   - 原因：身体疼痛

4. **母性攻击**：
   - 表现：保护幼崽
   - 原因：母性本能

#### 解决方案

1. **安全第一**：
   - 避免直接接触
   - 使用专业帮助
   - 必要时佩戴嘴套

2. **脱敏训练**：
   - 逐步接触刺激源
   - 正面强化
   - 不要惩罚

3. **就医检查**：
   - 排除身体疼痛
   - 检查甲状腺功能
   - 神经系统检查

## 基本训练

### 如厕训练

**步骤**：

1. **建立规律**：
   - 固定时间外出（饭后、睡醒后）
   - 每次到同一地点
   - 使用固定口令

2. **正面强化**：
   - 在正确地点排便后立即奖励
   - 热情表扬
   - 给予零食

3. **处理意外**：
   - 立即清理
   - 使用除臭剂
   - 不要惩罚（会增加焦虑）

4. **时间表**：
   - 幼犬每2-3小时一次
   - 成犬每4-6小时一次
   - 夜间可适当延长

### 基本指令训练

#### "坐下"训练

1. 手持零食靠近宠物鼻子
2. 慢慢将手向上移动
3. 宠物自然会坐下
4. 立即说"坐下"并奖励
5. 重复训练，逐渐减少零食

#### "等待"训练

1. 让宠物坐下
2. 手掌向前，说"等待"
3. 后退一步
4. 如果保持不动，立即奖励
5. 逐渐增加距离和时间

#### "过来"训练

1. 使用牵引绳
2. 说"过来"，轻拉牵引绳
3. 宠物靠近后立即奖励
4. 逐渐增加距离
5. 最终在无绳情况下训练

### 社会化训练

**关键期**：3-14周龄（犬）

**训练内容**：

1. **接触不同人**：
   - 成人、儿童、老人
   - 不同种族、性别
   - 戴帽子、穿制服的人

2. **接触不同动物**：
   - 其他狗
   - 猫
   - 小型动物

3. **接触不同环境**：
   - 街道、公园
   - 车内
   - 兽医诊所
   - 宠物店

4. **接触不同声音**：
   - 汽车喇叭
   - 雷声
   - 吸尘器
   - 烟花（录音）

## 行为训练要点

### 正面强化原则

1. **及时奖励**：
   - 行为发生后1-2秒内奖励
   - 让宠物明确关联

2. **一致性**：
   - 全家使用相同口令
   - 相同行为相同对待
   - 不要今天允许明天禁止

3. **短时高频**：
   - 每次训练5-15分钟
   - 每天2-3次
   - 在宠物精力充沛时训练

4. **循序渐进**：
   - 从简单到复杂
   - 逐步增加干扰
   - 不要急于求成

### 常见训练错误

1. **惩罚过度**：
   - 导致恐惧和焦虑
   - 破坏信任关系
   - 可能加重问题行为

2. **训练时间过长**：
   - 宠物注意力下降
   - 产生厌烦情绪

3. **口令混乱**：
   - 不同人使用不同口令
   - 口令过于复杂

4. **忽视身体语言**：
   - 尾巴摆动
   - 耳朵位置
   - 眼神
   - 身体姿态
